# LNK file
LNK files are used by the Windows OS to link one file to another. 
Metadata
- Location of the folder
- Date the LNK file was created
- Modified
- last accessed
- File size

Location: C:\Users\$USER$\AppData\Roaming\Microsoft\Windows\Recent
